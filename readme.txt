This plugin adds social network timelines to the template. These timelines are configured from the tab "edit" to the plugin itself. 

== Description  ==

At the moment, this plugin can only create and add Twitter timelines to the template.
The plugin creates a widget with the Twitter timeline. This plugin creates a data table that allows us to modify the timeline.
Furthermore, it creates a new principal menu element.
 
This plugin was not tested in previous version to 4.3.1. For this reason. we can not guarantee that it will work in previous versions. 

== Installation  ==

Steps to follow when installing the plugin:
 
1. Activate the plugin from "Plugins" section.
2. Select "Social Networks" option in the main menu.
3. Fill the form out and click "Save". The field "Css class" and "Data widget id" are required.
4. Add the widget to the "Sidebar Widget" of the template.
